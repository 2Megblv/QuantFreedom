# MultiAsset EA

Project folder for creating a Multi-Asset Expert Advisor (EA) for MetaTrader 5 (MT5).
The goal of this EA is to pass and run on Prop Firm accounts.

## Requirements
- Platform: MT5 (MQL5)
- Strategy: Multi-asset trading
- Goals: Pass prop firm challenges and manage live funded accounts

## Next Steps
1. Define the trading logic and indicators to be used.
2. Setup risk management rules (drawdown limits, position sizing) as required by Prop Firms.
3. Plan the multi-asset architecture (handling multiple symbols from a single chart or running on multiple charts).
